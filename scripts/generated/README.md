# Generated Configuration Files

This folder contains AWS resource IDs and configuration generated by the setup scripts.

## Files Created

| File | Created By | Contains |
|------|-----------|----------|
| `iam-config.json` | Phase 1 | Lambda execution role ARN |
| `s3-config.json` | Phase 2 | S3 bucket name and ARN |
| `cloudfront-config.json` | Phase 2 | Distribution ID, domain, URL |
| `cloudfront-bucket-policy.json` | Phase 2 | S3 bucket policy for CloudFront |
| `cognito-config.json` | Phase 3 | User Pool ID, Client ID, domain |
| `api-gateway-config.json` | Phase 4 | API ID, endpoint, authorizer ID |
| `lambda-config.json` | Phase 5 | Lambda function ARNs |
| `dynamodb-config.json` | Phase 6 | Table name and ARN |

## Important

- These files contain AWS account-specific IDs
- Do not commit these files to version control
- They are automatically regenerated by running setup scripts
- Used by `config.sh` to load AWS resource references
